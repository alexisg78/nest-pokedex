# Ejecutar en desarrollo

1. Clonar el repositorio

2. Ejecutar:
```
  npm install
```
3. Tener Nest CLI instalado
```
  npm i -g nestjs/cli
```
4. Levantar la Base de Datos
```
  docker-compose up -d
```
5. Reconstruir la Base de Datos con la semilla
